# GrowthikOS Design & Architecture Principles

## Platform Vision
GrowthikOS is a world-class AI-powered Business Operating System. It must feel like Apple, Vercel, Stripe, Linear, Raycast, and Cursor.
The experience must be modern, intelligent, calm, premium, and extremely fast.

## Design Principles
- Minimal, Clean, Enterprise, AI-first, Modern.
- Glassmorphism (used sparingly), soft gradients, large spacing, rounded corners.
- Micro interactions, beautiful typography, zero clutter.
- Responsive, accessibility-first (WCAG AA), keyboard friendly, touch friendly.

## User Experience
Every page answers: What can I do? What should I do next? How is my business performing? What needs attention? How can AI help?

## Tech Stack & Libraries
- **Core:** Next.js 16, React 19, TypeScript, Tailwind CSS v4, FastAPI, PostgreSQL, Redis.
- **UI:** shadcn/ui, Radix UI, Magic UI, Aceternity UI, Origin UI, 21st.dev inspired.
- **Animations:** Framer Motion (everywhere), GSAP (hero only), Lenis (smooth scroll). Use fade, scale, slide, parallax, magnetic buttons, animated borders, loading skeletons.
- **Charts:** Apache ECharts. Support all chart types, dark mode, exports.
- **Tables:** AG Grid Community. Virtualization, sorting, filtering, infinite scroll.
- **Maps:** Leaflet, MapLibre.
- **Editor:** Tiptap (slash commands, AI writing).
- **Workflow:** React Flow (nodes, edges).
- **Realtime:** FastAPI WebSockets.

## Features
- Command Palette (Ctrl+K) everywhere using cmdk.
- AI Copilot, Chat, Report Generator, Translation, Coding, Prompts.
- Comprehensive Report Builder (drag and drop).
- Multi-language support.

**FINAL GOAL:** Every screen should feel premium, interactive, animated, intelligent, and enterprise-ready. It is one connected AI Operating System, not a collection of pages.
